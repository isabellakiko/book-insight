## 分析方法

### 智能采样
均匀采样覆盖全书，快速建立人物脉络
- 适用于：初次分析、快速了解
- 章节数：30 章
- 耗时：~5 分钟

### 增量分析
在已有基础上继续分析更多章节
- 适用于：深入了解、补充细节
- 章节数：可配置（50-200）
- 耗时：视章节数而定

### 范围分析
针对特定章节范围深入分析
- 适用于：关键剧情、特定阶段
- 章节数：范围内所有出场章节
- 耗时：视范围大小而定
